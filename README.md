 Java 主控台遊戲合輯 (Console Game Collection)

這是一個使用 Java 語言開發的練習專案，整合了三款經典的文字介面小遊戲。本專案旨在練習 Java 基礎語法、資料結構（陣列）應用、邏輯演算法以及基礎的物件導向概念。

 🎮 遊戲內容
本專案包含以下三種遊戲模式：
 1. 打彈珠 (Marbles)
 玩法：玩家輸入想要投注的彈珠數量，並猜測 1-5 之間的數字。
 機制：系統會隨機產生結果。若猜中，玩家將獲得倍數獎勵；若猜錯，則扣除投注的彈珠。
 核心邏輯：條件判斷、數學運算。

 2. 抽鬼牌 (Lucky Card Draw)
 玩法：從 15 張蓋住的牌中（包含黑桃 A-K 與鬼牌）輸入數字進行抽牌。
 機制：只有抽中「鬼牌」才算獲勝，其餘皆為失敗。
 核心邏輯：
     陣列 (Array)：儲存撲克牌花色。
     洗牌演算法：實作「去重複」邏輯，確保每次遊戲的牌局排列皆為隨機且不重複。

 3. 蛇梯子 (Snakes and Ladders)
 玩法：支援雙人對戰（本機輪流操作），最先到達第 25 格者獲勝。
 機制：
     梯子：走到特定格子會自動前進（如第 3 格前進 8 格）。
     蛇身：走到特定格子會自動後退（如第 14 格後退 10 格）。
 核心邏輯：
     地圖設計：使用一維陣列的索引值 (Index) 代表地圖格數，陣列值代表位移量。
     物件導向：建立玩家類別來管理 ID 與狀態。

 🛠️ 技術運用
 程式語言：Java
 開發環境：JDK (Java Development Kit)
 核心觀念：
     基本 I/O：使用 `Scanner` 類別處理使用者輸入（特別處理 Big5 編碼）。
     流程控制：運用 `switch`、`if-else`、`for/while` 迴圈與標籤 (Label) 控制遊戲流程。
     資料結構：一維陣列 (1D Array) 的應用。
     物件導向 (OOP)：類別 (Class) 與物件 (Object) 的定義與實例化。
